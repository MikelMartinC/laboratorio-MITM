# Informe: Simulación de Ataque Man-in-the-Middle con Ettercap

En este laboratorio se realizó una simulación de un ataque Man-in-the-Middle (MITM) utilizando la herramienta Ettercap en Kali Linux, con el objetivo de comprender cómo un atacante puede interceptar el tráfico de red mediante técnicas de ARP poisoning. Para ello, se configuró Ettercap en modo gráfico, se seleccionó la interfaz de red adecuada y se escanearon los dispositivos conectados a la misma subred, identificando el gateway (router) y el equipo víctima. Posteriormente, se establecieron ambos como objetivos del ataque y se inició el envenenamiento ARP, logrando que el tráfico entre la víctima y el router pasara a través del equipo Kali. Finalmente, se verificó el funcionamiento del ataque accediendo a un sitio web de pruebas desde la máquina víctima, observando en Ettercap la captura de tráfico y credenciales enviadas, demostrando así la efectividad del ataque en un entorno controlado y resaltando la importancia de la seguridad en redes locales.
